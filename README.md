# Blog IA, Filosofia e Cultura Digital — Italo Marques Fabris

Site estático pronto para GitHub e Vercel.

## Estrutura

- `index.html` — página inicial do blog
- `artigos/` — páginas HTML separadas dos 15 artigos
- imagens carregadas por URLs do Pexels
- Bootstrap e Bootstrap Icons carregados via CDN

## Deploy na Vercel

1. Suba esta pasta para um repositório no GitHub.
2. Na Vercel, importe o repositório.
3. Framework: Other.
4. Build command: deixe vazio.
5. Output directory: deixe vazio ou use raiz do projeto.
